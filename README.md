## Usage

### Upload
```
  jaxx-upload -h
  jaxx [options]
    -b, --bucket [BUCKET]
    -k, --access-key [ACCESS_KEY]
    -s [ACCESS_SECRET],
    --access-secret
    -f, --file [FILE]
    -p, --privacy [PRIVACY]
    -h, --help
```
